DLしてくださり､ありがとうございます。
『Fate/stay night』よりギルガメッシュです。

ドットは『Fate/unlimited codes』のギル様をトレースしたものですが、
性能は完璧にＵＣ再現というわけではないので注意。

まだドット９割ぐらい下書きですが、楽しんでいただければ幸いです。

このキャラはエフェクトやヘルパーを多く使います。
mugen.cfgの上限を多めに設定することを推奨します。

------------ボタン配置-------------------------------------------------------------

・a　…　弱攻撃

・b　…　中攻撃

・c　…　強攻撃

・z　…　リフレクトガード

・start　…　挑発

------------必殺技コマンド---------------------------------------------------------

・冥府の門(空中可)　　　…　236 + a OR b

・天の鎖・縛　　　　　　…　623 + a

・天の鎖・引　　　　　　…　623 + b

・氷の短剣　　　　　　　…　214 + a

・無銘・螺旋剣　　　　　…　214 + b

・鎌　　　　　　　　　　…　214 + c

------------超必殺技コマンド-------------------------------------------------------

・王の財宝Ⅰ(空中可)　　　…　236 + c
　　[1ゲージ消費]

・天の鎖・エルキドゥ　　　…　623 + c
　　[2ゲージ消費]

・天地乖離す開闢の星　　　…  41236 + c
　　[3ゲージ消費]

・王の財宝Ⅱ　　　　　　　…　22 + c (タメ可)　　　　　※本気状態時のみ


・天地波涛す終局の刻　　　…　214214 + c　　　　　 　　※本気状態時のみ
　　[3ゲージ消費]

------------その他コマンド---------------------------------------------------------

・投げ　　　　　　…　6x OR 4x

・空中ダッシュ　　…　66 OR 44

・ハイジャンプ　　…　28

・本気状態　　　　…　x　　　　　　　　　　※本気ゲージがMAX時のみ

・本気バースト　　…　x　　　　　　　　　　※本気ゲージがMAXで､くらいorガード中の時

・避け　　　　　　…　2x

・慢心　　　　　　…　2start
