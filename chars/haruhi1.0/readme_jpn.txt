こなたはパワー消費0で呼べます。また出した本によって効果が変わります。
コンプティーク…パワーが100増えます。
ニュータイプ…パワーが200増えます。

なし…パワーが400減ります。

●AI設定
　config.txtを開き[State -1, AI config]の最後のvar(59)を変えることでAIの動きが若干変わります。
(59) =1で対人向け
(59) =2で対AI戦向きになります。
対人戦目的の人は投げなど切り返しの反応が弱くなる(59) =1を推奨します。

●デフォルトカラーの下着の色変更
　Haruhi.defをメモ帳などで開き、Pal4を3種類から選んでください。mintが「孤島症候群」のミントグリーン、orangeが「憂鬱」のオレンジ、whiteが白になります。

●部室連れ込みについて
相手がブリス対応だと、ゲージ2本時にその衣装で放り出されます。SOSアタックでも相手がブリス絵になります。
airに[Begin Action 9020]があると対応するので他の絵が入っていると不自然な表示になることがあります。

●脱衣KO演出の選択
　敗北時(2本先取なら2敗目で)にKOされると特殊なやられになります。状況によって吹っ飛びとへたり込みのどちらかになります。この演出にしたい人は、config.txtを開き、0の前に;をつけて、1の前の;を削除してください。
;Stripping KO ON(1)/OFF(0)
[State -1, setting]
type = varset
trigger1 = 1
v = 20
value = 0;off
;value = 1;on

●敗北時におもらしするのが見たい人はconfig.txt(WinMugenの人はconfig_win.txt)を開き、同様の処理をしてください。脱衣せずにおもらしだけ見るのも可能です。ONにすると、オシオキでKOした相手もおもらしするので注意。
;Urination ON(1)/OFF(0)
[State -1, setting]
type = varset
trigger1 = 1
v = 23
value = 0;off
;value = 1;on

●敗北時に触手による特殊KOが見たい人はconfig.txt(WinMugenの人はconfig_win.txt)を開き、同様の処理をしてください。
;ryona KO ON(1)/OFF(0)
[State -1, setting]
type = varset
trigger1 = 1
v = 27
;value = 0;off
value = 1;on

●読まなくてもいい駄文
「私のキャラがなければ作ればいいのよっ！」
　たまたまアニメを見ていた程度でハルヒに興味なかった私が、気がついたらキャラ製作をしているとは…ハルヒに巻き込まれたキョンの気分です。
　製作コンセプトは、ストライカー呼びまくるうざいキャラ。ハルヒ自身は単なる腕っ節の強い乱暴者で仲間がうまくフォローして勝つけど、ハルヒは自分の力で勝っていると思っている設定です。
　一部のドット改変程度の経験しかない私が、まるまる1体打つのは無謀でしたが、なんとか形になりました。くじける前にキャラにしたかったので短期間で枚数を用意するため、ドット絵はイメージに近いポーズのメルブラのキャラを混ぜ合わせて改変しています。改めてゼロから打つドッターの方を尊敬します。各リョナキャラのやられに対応していますが何に対応しているかは各自探してください。意味が分からない人は気にしないでください。
　音声は様々なアニメやゲームから集めました。なるべくBGMがかぶってないセリフを選んだのでハルヒっぽくない口調もありますがご了承ください。

　MUGENって何？どうすればハルヒが使えるの？といった超初心者の方は、まず検索ツールでMUGENを調べてください。丁寧に解説しているサイトが多数あります。MUGENはゲームではありません。自分でカスタマイズできるゲーム製作ツールです。設定に勉強と努力が必要なのは覚悟してください。
　俺ならもっと良いハルヒ作れるぜって人は、ぜひ作って見せてください。私も励みになります。
